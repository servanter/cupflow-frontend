## 🔐 3. 认证请求流程

```
┌─────────────────────────────────────────────────────────────┐
│  页面发起需认证 API 调用                                      │
│  const res = await api.get('/api/user/profile', true)       │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ↓
         ┌───────────────────────────────┐
         │  request() 方法被调用          │
         │  options = {                   │
         │    url: '/api/user/profile',  │
         │    method: 'GET',             │
         │    needAuth: true             │
         │  }                            │
         └────────────┬──────────────────┘
                      │
                      ↓
         ┌────────────────────────────┐
         │  构建 HTTP 请求头           │
         │  {                          │
         │    'Content-Type':          │
         │    'application/json'       │
         │  }                          │
         └────────────┬─────────────────┘
                      │
                      ↓
         ┌────────────────────────────────┐
         │  检查 needAuth 参数              │
         │  if (options.needAuth)          │
         └────────────┬───────────────────┘
                      │
              ┌───────┴────────┐
              │                │
          是 ↓              否 ↓
         ┌────────────┐  ┌──────────┐
         │获取 token  │  │跳过认证  │
         │getToken()  │  └──────────┘
         └─────┬──────┘       │
               │              │
               ↓              │
         ┌──────────────────┐ │
         │uni.getStorageSync│ │
         │('user_token')    │ │
         └──────┬───────────┘ │
                │              │
                ↓              │
         ┌────────────────────┐
         │ token 存在?         │
         │                    │
         └───┬────────────┬──┘
             │            │
          是 │         否 │
             ↓            ↓
        ┌─────────┐   跳过(不添加)
        │添加请求 │
        │头:      │
        │Bearer   │
        │{token}  │
        └────┬────┘
             │
             └──────┬───────┐
                    │       │
                    ↓       ↓
         ┌──────────────────────────┐
         │  发送 uni.request()      │
         │  url: BASE_URL + url     │
         │  method: 'GET'           │
         │  data: undefined         │
         │  header: {Authorization} │
         └────────────┬─────────────┘
                      │
                      ↓
         ┌────────────────────────────────┐
         │  后端返回响应                    │
         │  {                              │
         │    "code": ...,                │
         │    "message": ...,            │
         │    "data": {...}              │
         │  }                            │
         └────────────┬───────────────────┘
                      │
                      ↓
         ┌─────────────────────────────────┐
         │  检查响应码                      │
         │  if (res.code === 401)         │
         └────────┬──────────────┬────────┘
                  │              │
             401 ↓          其他 ↓
         ┌─────────────────┐ ┌─────────┐
         │未登录/已过期    │ │成功     │
         │                 │ │         │
         │navigateTo(      │ │resolve( │
         │  '/pages/      │ │  data)  │
         │  login/        │ │         │
         │  index')       │ │         │
         │                 │ │         │
         │reject(Error)   │ │         │
         └─────────────────┘ └────┬────┘
                 │                 │
                 └────────┬────────┘
                          │
                          ↓
                  页面收到响应数据
```
